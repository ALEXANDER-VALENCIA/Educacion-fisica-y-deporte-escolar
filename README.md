# Educacion-fisica-y-deporte-escolar
Repositorio académico con materiales de Educación Física y Deporte para primaria y secundaria.
## Objetivo
Apoyar el proceso de enseñanza–aprendizaje mediante recursos didácticos, planeaciones y actividades que promueven el desarrollo motor, la actividad física, el deporte escolar y hábitos de vida saludable.
## Contenido
- Planeaciones de clase
- Proyectos pedagógicos
- Recursos didácticos
## Niveles
- Educación Primaria
- Educación Secundaria
